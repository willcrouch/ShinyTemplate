# ShinyTemplate
Template of a Shiny Dashboard
This includes a lot of options and interactions as a whole Shiny app and not just code snippets

Widgets in the template:
1. Shiny Dashboard
  a. UI
  b. Server
  c. Global
2. Dropdowns
  a. Menu Outputs
  b. Menu
  c. Generated values
  d. Generated messages
3. Tabs
4. Boxes
5. Infoboxes
6. Text
  a. UI generated
  b. Server generated
  c. HTML
7. Leaflet
